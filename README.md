# IllustrationWebsite
## 一、介紹

繪圖創作網站，使用者能在上面分享自己的繪圖創作，並透過標籤來讓更多人看到自己的作品，除作品作者外，其他的使用者也能添加新的標籤到作品上。
觀看他人作品時，使用者能對作品按讚並留言表達自己的想法。

為前後端分離的做法。
    * 前端資料於－Illus.Client  使用React和TypeScript編寫  
    * 後端資料於－Illus.Server  使用.NET編寫

## 二、運行環境

### 前端
    * Node.js 21.7.1
### 後端
    * .NET 8.0 
    * MSSQL 18
